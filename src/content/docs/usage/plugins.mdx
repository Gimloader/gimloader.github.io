## Official Plugins

Gimloader offers a variety of official plugins which are guranteed to be safe. These can be easily downloaded within the `plugins` tab of the Gimloader menu by clicking the "Official Plugins" button. You can also browse them [here](/plugins), or using the sidebar on the left.

## Unofficial Plugins

Unofficial plugins can potentially be unsafe and risk your account, so take some caution when installing them.

### Installing with a link

If an install link is provided with a plugin the plugin can be installed by simply clicking the link and pressing "install". Install links will take the form *gimloader.github.io/install/?installUrl=...* Plugins can also be installed by selecting "Install from URL" in the `Create/Install` dropdown.

### Importing a file

Plugin files can be dragged and dropped onto the `plugins` tab of the Gimloader menu to install them. You can also upload them manually like so:

<Steps>
1. Download the plugin's file

2. Open the Gimloader UI in Gimkit by pressing a wrench button or Alt + P

3. Select "Upload File" from the `Create/Install` dropdown

4. Upload the plugin's file
</Steps>
