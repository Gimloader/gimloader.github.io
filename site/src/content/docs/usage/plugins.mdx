You can find official plugins in the sidebar on the left, which have a button on the page to easily install them.

## Installing with a link

If an install link is provided with a plugin the plugin can be installed by simply clicking the link and pressing "install". Install links will take the form *thelazysquid.github.io/Gimloader/install/?installUrl=...*

## Importing a file

<Steps>
1. Download the plugin's file

2. Open the Gimloader UI in Gimkit by pressing a wrench button or Alt + P

3. Click the upload icon in the top left

4. Upload the plugin's file
</Steps>
