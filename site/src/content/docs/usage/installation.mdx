---
title: Installing Gimloader
description: A step by step guide on how to install Gimloader
---

import { Steps } from "@astrojs/starlight/components";

<Steps>
1. Install [Violentmonkey](https://violentmonkey.github.io) for your browser.

2. Click on [this link](https://raw.githubusercontent.com/TheLazySquid/GimLoader/main/build/bundle.user.js) and it install.

3. Confirm Gimloader is working by checking whether the wrench icon appears next to the join button on the [Gimkit join page](https://gimkit.com/join).
</Steps>
